# 📊 可视化交易模拟器使用指南

## 🚀 快速开始

### 启动 UI
```bash
npm run ui
```

访问：**http://localhost:5173**

## 🎮 界面功能

### 1. 顶部栏
- 左侧：项目标题 💹 USD/JPY 交易模拟器
- 右侧：当前时间显示（随 K 线回放实时更新）

### 2. 左侧区域 - K 线图表

#### 图表功能
- **专业 K 线图**：TradingView Lightweight Charts 渲染
- **颜色说明**：
  - 🟢 绿色：上涨 K 线
  - 🔴 红色：下跌 K 线
- **交易标记**：
  - 实线：入场价格（绿色=做多，红色=做空）
  - 虚线：止损/止盈线
  - 橙色线：出场价格

#### 播放控制面板
```
⏮️ 重置  ▶️ 播放  速度: [1x]  进度: 120 / 4260 (2.8%)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

| 控件 | 功能 |
|------|------|
| ⏮️ 重置 | 回到开始，清除所有交易 |
| ▶️ 播放 | 开始回放（每分钟一根 K 线） |
| ⏸️ 暂停 | 暂停回放 |
| 速度下拉 | 0.5x, 1x, 2x, 4x, 10x |
| 进度条 | 拖动跳转到任意时间点 |

### 3. 右侧区域 - 交易面板

#### 当前价格显示
```
当前价格: 152.500
```

#### 交易方向选择
```
[📈 做多]  [📉 做空]
```
- 做多：预期价格上涨
- 做空：预期价格下跌

#### 参数设置

**持仓时间（分钟）**
```
[60] 分钟
```
- 设置交易持续多久
- 范围：1 - 1440 分钟

**止损设置**
```
☑️ 使用止损
   [50] pips → (152.000)
```
- 勾选启用止损
- 输入止损点数
- 自动计算止损价格
- 做多：止损在入场价格下方
- 做空：止损在入场价格上方

**止盈设置**
```
☑️ 使用止盈
   [100] pips → (153.000)
```
- 勾选启用止盈
- 输入止盈点数
- 自动计算止盈价格
- 做多：止盈在入场价格上方
- 做空：止盈在入场价格下方

**仓位大小**
```
[1] 手
```
- 标准手：1 手 = 100,000 单位
- 每 pip 价值 = $10

#### 开始交易按钮
```
[🚀 开始交易]
```

## 📝 使用流程

### 完整操作步骤

1. **加载页面**
   - 自动加载示例数据（约 4260 根 K 线）
   - 显示数据时间范围

2. **选择起始时间**
   - 方式 A：直接从第一根 K 线开始
   - 方式 B：拖动进度条到想要的时间点

3. **设置交易参数**
   ```
   方向: 📈 做多
   持仓: 60 分钟
   止损: 50 pips
   止盈: 100 pips
   仓位: 1 手
   ```

4. **点击开始交易**
   - 面板切换到"交易进行中"状态
   - 图表上绘制入场线和止损/止盈线

5. **点击播放按钮**
   - K 线开始逐分钟回放
   - 价格实时更新
   - 自动检测止损/止盈触发

6. **观察交易进程**
   ```
   🔥 交易进行中
   方向: 📈 做多
   入场价格: 152.500
   当前价格: 152.600
   止损: 152.000
   止盈: 153.000
   仓位: 1 手
   ```

7. **查看交易结果**
   ```
   ✅ 盈利
   损益: $100
   点数: +10 pips
   百分比: +0.0655%
   出场价格: 153.000
   持仓时间: 45 分钟
   出场原因: 🎯 触发止盈
   ```

## ⚠️ 注意事项

1. **数据范围**
   - 确保持仓时间不超过数据范围
   - 建议持仓时间 ≤ 120 分钟

2. **止损止盈设置**
   - 做多：止损 < 入场价 < 止盈
   - 做空：止盈 < 入场价 < 止损

3. **播放状态**
   - 交易开始后需要点击播放
   - 播放中可以暂停
   - 重置会清除所有交易

4. **性能**
   - 数据量大时（10000+ K 线）可能略卡
   - 建议使用示例数据（4260 根）

## 🆘 故障排查

### 问题：图表不显示
- 检查浏览器控制台是否有错误
- 确认数据文件已加载
- 刷新页面

### 问题：播放没反应
- 确认已点击"开始交易"
- 确认进度条不在末尾
- 检查是否已暂停

### 问题：止损止盈不触发
- 检查价格设置是否合理
- 确认 K 线正在播放
- 查看图表上的标记线

### 问题：损益计算不准确
- 验证仓位大小设置
- 检查入场和出场价格
- USD/JPY: 1 pip = 0.01, 1 手 = $10/pip
